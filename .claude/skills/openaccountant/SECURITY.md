# Security Policy

## About This Repo

This repository contains SKILL.md files — markdown instruction files, not executable code. Security risks are limited to malicious skill content that could instruct an AI agent to take harmful actions.

## Reporting a Concern

**Do NOT open a public GitHub issue for security concerns.**

Email **security@openaccountant.ai** with:

- The skill file in question
- Description of the concern
- Potential impact

### What to expect

- **Acknowledgment** within 48 hours
- **Assessment and action** within 7 days
